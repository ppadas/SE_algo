https://codeforces.com/contest/1631/problem/A

Даны два массива a и b, содержащие n положительных целых чисел каждый. Вы можете выполнить следующую операцию неограниченное количество раз:

Выбрать индекс i (1≤i≤n) и поменять местами ai и bi (то есть ai становится равным bi и наоборот).
Найдите минимальное возможное значение выражения max(a1,a2,…,an)⋅max(b1,b2,…,bn), которое можно получить, проделывая описанную операцию любое количество раз (возможно ноль).

Входные данные
Каждый тест состоит из нескольких наборов входных данных. Первая строка ввода содержит одно целое число t (1≤t≤100) — количество наборов входных данных. Далее следует их описание.

Первая строка каждого набора содержит целое число n (1≤n≤100) — размер массивов.

Вторая строка каждого набора содержит n целых чисел a1,a2,…,an (1≤ai≤10000), где ai — i-й элемент массива a.

Третья строка каждого набора содержит n целых чисел b1,b2,…,bn (1≤bi≤10000), где bi — i-й элемент массива b.

Выходные данные
Для каждого набора входных данных напечатайте одно целое число — минимальное возможное значение выражения max(a1,a2,…,an)⋅max(b1,b2,…,bn), которое можно получить, применяя описанную операцию любое количество раз (возможно ноль).
